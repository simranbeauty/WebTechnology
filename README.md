# Web Technology Lab Reports

This repository contains the source code and lab reports for the Web Technology course. The work is organized into different branches, each corresponding to a specific lab unit.

## Branches
- `lab-unit-1`: HTML & CSS
- `lab-unit-2`: JavaScript & Client-Side Scripting
- `unit3`: PHP & Server-Side Scripting

Please switch to the respective branch to view the files and reports for each lab.
